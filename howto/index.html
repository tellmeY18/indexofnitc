How To Contribute⌗

The Projects aims to be a completely community driven initiative that will move according to the demand and usage of the community. The Source code of the project is available Here and is built using Hugo. The indexes are categorised and are added to markdown files with suitable names. They are located inside the ‘posts’ directory which is inside the ‘content’ directory.

You can click on the markdown file which you are interested to contribute to and click on the pencil button that is marked in the figure below to edit the content.

Observe the data given and add the links using the same format. You can also refer to the Markdown Cheatsheet to know more on how to manage markdown files. Press on the commit button to submit your entries.
